# 해커톤 매뉴얼

이 문서는 해커톤을 개최하고자 하는 모든 조직들이 참고할 수 있는 매뉴얼입니다.

해커톤은 재미있어야 합니다. 해커톤은 다양성이 있어야 합니다. 
해커톤은 안전해야합니다. 해커톤은 준비가 잘 되어야 합니다.

그런 해커톤을 만드는데 도움이 되면 좋겠습니다.

## [진짜 매뉴얼은 manual.md 를 클릭](manual.md)
